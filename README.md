# Cryptography & Secure Password Manager

## Overview

Cryptography & Secure Password Manager is a simple Python terminal-based project that stores and manages passwords using basic encryption and decryption techniques.

The project uses a Caesar Cipher style encryption system where letters are shifted by a fixed value before storing passwords.

This project was made to practice Python fundamentals and basic cryptography concepts.



## Features

* Store passwords for different accounts
* Encrypt passwords before saving
* Decrypt passwords when viewing
* Search saved account passwords
* View all stored accounts
* Menu-driven terminal interface

## Encryption Method

The project uses a simple Caesar Cipher encryption technique.

Example:

```text id="jl1pt4"
Original Password : hello123
Encrypted Password : khoor123
```

Each alphabet letter is shifted by `+3`.


## Menu Options

1. View Saved Accounts
2. Add New Password
3. Search Account Password
4. View Decrypted Passwords
